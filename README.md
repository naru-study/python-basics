## 🚀 始め方

```bash
git clone https://github.com/YOUR_NAME/python-basics.git
cd python-basics
```

各フォルダに移動して README.md を読み、`answer.py` を自分で作成してください。

## ⚠️ルール
- ブランチを1問ずつちゃんと切ること
- プルリクエストを出す際は、ikemuraを指定
- プルリクエストの時に、どこを見て欲しいか、しっかり記述する
- readmeに解答例が記載してあるが、まずは見ないで解くこと
- プルリク出したらMattermostで連絡
